SISTEMA DE CONTROLE DE TRABALHOS DE CONCLUSÃO DE CURSO
================
Trata-se de um sistema web que permite a automatização do processo de avaliação de TCC dos alunos do Instituto Federal do Maranhão (IFMA.

Tecnologias:
-------------
    * JSF 2.3
    * Spring Boot 2.2.0
    * Primefaces 6.2
    * Maven
    * Wildfly 18
    * PostgreSQL 12
